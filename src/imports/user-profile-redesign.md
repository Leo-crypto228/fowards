PROMPT – REFONTE INTERFACE PROFIL UTILISATEUR (FF)

Je veux modifier la présentation des informations et l’interface du profil utilisateur dans FF.

Le design doit maintenant s’inspirer du profil d’une communauté, pour créer une cohérence visuelle dans toute l’application.

Le style général doit rester :

moderne

minimaliste

mobile-first

avec des surfaces liquid glass / glassmorphism.

1. Structure du haut du profil

La présentation doit être similaire à celle des profils de communauté.

Bannière

En haut de la page :

une bannière horizontale pleine largeur

Photo de profil

La photo de profil doit être :

positionnée à gauche

située à environ 2/3 dans la bannière

2 fois plus grande que la taille actuelle.

Elle doit légèrement dépasser vers le bas de la bannière.

2. Informations principales

Sous la bannière :

Nom utilisateur

Afficher :

le nom ou @nom de l’utilisateur

Mot unique de description

Sous le nom :

Créer un boudin blanc (capsule) avec :

texte noir

Contenu :

un mot qui décrit l’utilisateur

Exemple :

Builder

Visionnaire

Athlete

Explorer

Description utilisateur

Sous ce boudin :

Afficher la description du profil.

Si la description est trop longue :

afficher seulement une partie

ajouter “plus…”

cliquer dessus doit déployer le texte complet.

Cela évite que la description prenne trop de place.

3. Abonnés et abonnements

Sous la description :

Afficher :

Nombre d’abonnés
Nombre d’abonnements

Exemple :

125 abonnés • 80 abonnements

Règles :

les nombres doivent être en gras

cliquer sur abonnés ouvre la liste des abonnés

cliquer sur abonnements ouvre la liste des comptes suivis.

4. Streak

Dans le coin droit sous la bannière, afficher :

la streak de l’utilisateur.

Instruction importante :

Utiliser exactement le même design de streak que celui utilisé dans :

les posts

les communautés

les autres zones de l’application.

Cela permet de garder une cohérence visuelle globale.

5. Hashtags du profil

Sous la zone abonnés / abonnements :

Afficher les hashtags du profil.

Style :

texte blanc / gris

affichage horizontal

exemple :

#entrepreneur
#buildinpublic
#fitness
#discipline

6. Fin de la surface d’information

Toute cette zone doit être contenue dans une surface liquid glass grise.

Cette surface représente la carte d’information principale du profil.

Après les hashtags :

→ la surface liquid glass se termine.

7. Bloc statistiques personnelles

Sous un petit espace :

Créer une nouvelle surface avec bords arrondis et effet liquid glass.

Cette surface contient trois statistiques :

Constance

Score de constance de l’utilisateur.

Objectifs accomplis

Nombre d’objectifs terminés.

Exemple :

12 objectifs accomplis.

Jours sur FF

Nombre de jours depuis l’inscription.

Exemple :

78 jours sur FF.

8. Objectif actif

Sous les statistiques :

Titre :

Objectif actif

Sous le titre :

Afficher :

le nombre de jours depuis le début de cet objectif.

Nom de l’objectif

Sous cela :

Afficher :

le nom de l’objectif.

Description de l’objectif

Ajouter une description courte.

Si la description est trop longue :

→ afficher “plus…”

Barre de progression

Sous la description :

Ajouter la barre de progression utilisée ailleurs dans l’application.

Design :

capsule / boudin

fond gris

progression violette.

9. Évolution

Sous l’objectif actif :

Titre :

L’évolution

Sous ce titre :

Créer trois boudins de sélection.

Style :

bords gris

texte blanc.

Les trois filtres sont :

Depuis toujours

30 derniers jours

Cette semaine

Diagramme

Sous ces filtres :

Afficher un diagramme d’évolution.

Le graphique doit montrer :

la progression de l’utilisateur

avec un pourcentage visible.

Style :

minimaliste

courbe violette

cadre gris arrondi.

10. Badges

Sous un petit espace :

Créer une nouvelle surface liquid glass avec coins arrondis.

Titre :

Badges obtenus

Afficher plusieurs badges gagnés par l’utilisateur.

Exemples possibles :

Premier objectif accompli

7 jours de streak

30 jours de streak

100 messages

Explorateur de tribus

Contributeur actif

Builder constant

Les badges doivent être :

visuels

motivants

faciles à comprendre.

11. Objectif UX

Le profil doit transmettre immédiatement :

la progression

la constance

l’engagement

la personnalité de l’utilisateur.

Le design doit donner envie aux autres utilisateurs :

de suivre la personne

de regarder son évolution

de participer à la communauté FF.
