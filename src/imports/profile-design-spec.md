Tu es un Senior Product Designer spécialisé en interfaces mobiles premium iOS nouvelle génération.

Ta mission : concevoir la page PROFIL d’une application appelée FF.

FF n’est PAS un réseau social classique.
FF est une application de progression personnelle, discipline et trajectoire long terme.

━━━━━━━━━━━━━━━━━━━━━━
🎨 DIRECTION ARTISTIQUE OBLIGATOIRE
━━━━━━━━━━━━━━━━━━━━━━

• Style iOS moderne next-gen.
• Formes très arrondies, organiques.
• Cartes flottantes avec coins ultra arrondis.
• Effet liquid glass / glassmorphism subtil (blur, transparence maîtrisée, reflets légers).
• Ombres profondes mais douces.
• Dégradés fluides et lumineux (bleu électrique, violet profond, orange chaud, rose vibrant).
• Couleurs joyeuses mais premium, jamais enfantines.
• Effet glow léger sur les éléments importants (progression, bouton principal).
• Typographie moderne type SF Pro / Inter / Neue Haas Grotesk.
• Hiérarchie typographique forte (gros titres nets, sous-texte discret).
• Interface respirante avec beaucoup d’espace négatif.
• Sensation cockpit personnel + énergie maîtrisée.

L’écran doit donner une sensation :
Discipline.
Obsession saine.
Clarté.
Progression visible.
Énergie structurée.

━━━━━━━━━━━━━━━━━━━━━━
📱 STRUCTURE COMPLÈTE DU PROFIL
━━━━━━━━━━━━━━━━━━━━━━

1️⃣ HEADER IDENTITAIRE (impact émotionnel)

Fond :
Gradient liquide dynamique (subtil mouvement visuel suggéré).
Effet glass léger.

Contenu :
• Photo de profil ronde (légèrement surélevée avec ombre douce).
• Nom complet en bold.
• Phrase courte identitaire (ex : “Obsédé par la progression.” / “Mission en cours.”).
• Ligne secondaire discrète :
  “Depuis 143 jours sur FF”

Ambiance :
On doit sentir la trajectoire dès les 3 premières secondes.

━━━━━━━━━━━━━━━━━━━━━━
2️⃣ BLOC OBJECTIF ACTIF (ÉLÉMENT CENTRAL)

Carte glass arrondie, flottante.

Contenu :
• Titre : “Objectif actif”
• Intitulé clair de l’objectif.
• Barre de progression horizontale brillante avec léger glow.
• % affiché en grand (ex : 63%).
• Valeur actuelle / valeur cible.
• Date de début.
• Ligne “Restant : XXX”.

Micro-message dynamique sous la barre :
- Si progression récente : “Vous avancez.”
- Si stagnation : “Prochaine étape ?”
- Si baisse : “Réajustez. Continuez.”

Cette carte doit être visuellement dominante.

━━━━━━━━━━━━━━━━━━━━━━
3️⃣ BLOC DISCIPLINE & CONSTANCES

Disposition en 2 ou 3 mini-cartes arrondies.

Carte 1 :
🔥 Streak actuel
“27 jours consécutifs”

Carte 2 :
📊 Score de constance
“86% cette semaine”

Carte 3 :
🎯 Objectifs complétés
“4 terminés”

Chaque carte :
• Fond glass léger
• Icône minimaliste
• Couleur d’accent différente subtilement
• Ombre douce
• Design rond

━━━━━━━━━━━━━━━━━━━━━━
4️⃣ GRAPHIQUE D’ÉVOLUTION (30 JOURS)

Carte large arrondie.

Graphique :
• Ligne fluide.
• Glow léger sur la ligne.
• Axe horizontal = temps.
• Axe vertical = métrique principale.
• Lisible, pas technique.
• Style moderne, minimal.

But :
Visualiser la trajectoire.
Créer une sensation de montée progressive.

━━━━━━━━━━━━━━━━━━━━━━
5️⃣ INDICE D’ENGAGEMENT / INTENSITÉ

Carte circulaire glass.

Contenu :
• Score global (ex : 78%).
• Petit cercle progressif dynamique.
• Couleur évolutive selon performance.
• Label : “Indice d’engagement”

Doit transmettre énergie maîtrisée.

━━━━━━━━━━━━━━━━━━━━━━

Cartes flottantes :
• Titre

Uniquement progression.
