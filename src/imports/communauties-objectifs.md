PROMPT – SECTION OBJECTIFS (COMMUNAUTÉS FF)

Je veux ajouter une troisième section dans l’interface des communautés dans l’application FF.

Actuellement la navigation comporte :

Actus

Discussion

Il faut ajouter une troisième section appelée : Objectifs.

La barre de navigation devient donc :

Actus – Discussion – Objectifs

1. Accès à la page Objectifs

Quand l’utilisateur clique sur Objectifs, il arrive sur une nouvelle page dédiée aux objectifs de la communauté.

En haut de l’écran :

un petit bouton "Retour" en haut à gauche

ce bouton permet de revenir à la page profil de la communauté

Le bouton retour doit utiliser l’animation visuelle "glace / liquide / glace" déjà utilisée ailleurs dans l’application pour la cohérence du design.

2. En-tête de la page

Au centre de l’écran se trouve un bloc d’introduction.

Ce bloc est placé dans une surface grise sombre (gris presque noir) avec des coins légèrement arrondis.

Dans ce bloc :

Titre principal (au centre) :

Objectifs – [Nom de la communauté]

Exemple :

Objectifs – Entrepreneurs
Objectifs – Fitness
Objectifs – IA

Sous ce titre, légèrement plus petit :

Progression visible

Cette phrase sert de petite accroche inspirante.

Sous ce bloc, la page redevient fond noir pour afficher les objectifs.

3. Objectif n°1 – Croissance de la communauté

Premier objectif affiché :

Titre

100 membres

Description

Un petit texte expliquant l’objectif.

Exemple :

Atteindre 100 membres actifs dans cette communauté.

Barre de progression

Sous la description :

Une barre de progression horizontale.

Design :

forme longue et arrondie (style capsule / boudin)

fond gris sombre

à l’intérieur une barre violette qui avance selon la progression

La barre violette doit :

prendre exactement la place correspondant au pourcentage

rester dans la forme du boudin

Exemple :

20% → la barre violette remplit 20% du boudin

80% → elle remplit 80%

Couleur de la progression :

violet (couleur principale de FF)

Courbe d’évolution

Sous la barre de progression :

Un diagramme avec une courbe montrant l’évolution depuis le début.

Cette courbe représente par exemple :

la croissance du nombre de membres dans le temps.

Design du graphique :

placé dans un cadre gris sombre

coins arrondis

courbe violette

fond discret

style minimaliste et moderne

4. Espace de séparation

Après le premier objectif :

Ajouter un espace vertical visible pour bien séparer les objectifs.

5. Objectif n°2 – Activité de la communauté

Titre :

Parler, évoluer tous les jours

Description :

Un texte expliquant l’objectif.

Exemple :

Encourager les membres à échanger et progresser ensemble chaque jour.

Barre de progression

Même design que l’objectif précédent :

boudin gris

progression violette

Cette barre représente par exemple :

Nombre de messages quotidiens

Exemple :

Objectif :

50 messages par jour

La progression se remet à zéro chaque jour et indique le pourcentage atteint.

Courbe d’évolution

Sous la barre :

Même type de graphique que le premier objectif.

Le graphique montre par exemple :

l’évolution du nombre de messages par jour.

Design :

cadre gris

coins arrondis

courbe violette

style minimaliste.

6. Objectif UX global

Cette page doit transmettre une sensation de :

progression collective

motivation

énergie de groupe

Les membres doivent voir :

la croissance de la communauté

l’activité quotidienne

leur impact sur les objectifs

La page doit être :

claire

visuelle

motivante

moderne

mobile-first.
