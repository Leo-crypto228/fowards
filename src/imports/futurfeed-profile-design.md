🎯 MASTER PROMPT — PROFIL FF (Version Premium Liquid Glass)

You are a Senior iOS Product Designer.

Design the mobile iOS-first Profile screen for a platform called “FuturFeed (FF)”.

FF is NOT a traditional social network.
It is a public progression and discipline platform.

The design must evoke:

Pride

Stylish discipline

Forward movement

Mental clarity

The overall vibe must feel close to the premium ecosystem of Apple UI principles:
round → fluid → modern → premium
No aggressive shapes. No rigid layout.

🌑 GLOBAL THEME

Dark global theme (deep charcoal / near-black background, not pure black)

Background must use a subtle gradient, not flat color

High contrast between background and glass components

Atmosphere: focused, ambitious, clean

Accent colors allowed (used only for progression and emphasis):
Red, Blue, Yellow, Orange, Green, Purple, Pink
(we will test variations later)

Never use all colors at once randomly — colors must signal progression or achievement.

🧊 LIQUID GLASS SYSTEM (MANDATORY)

All main cards must use a premium layered liquid glass effect.

For each glass card:

Border radius: 24px minimum

Background fill: white at 10–14% opacity

Background blur: 22–28px

Border: 1px solid white at 18% opacity

Inner top highlight gradient: white → transparent, 12% opacity

Ambient shadow:

Y offset: 10px

Blur: 28px

Opacity: 10%

Soft gray, not pure black

Must create real depth.
No flat blur.
No cheap glass look.
No frosted rectangle effect.

Glass must feel like layered translucent material with light interaction.

📐 PROFILE STRUCTURE

The profile screen must visually communicate progression immediately.

1️⃣ HEADER ZONE (Hero Section)

Contains:

Profile picture (circular, 90–110px)

Name (large, dominant typography)

Subtitle / mission (smaller, subtle)

Typography:

Main name size: 25

Secondary text: 15

One single modern sans-serif font only

Strong hierarchy contrast between major and minor text

Below the name:
A primary progression indicator (must be visually powerful):

Circular level indicator OR

Progress ring around profile image

Use one accent color dynamically for the current level.

2️⃣ OBJECTIVE / FORWARD SECTION

Glass card showing:

Current main goal

Percentage progress

Visual bar progression

Progress bar:

Rounded edges

Height: 10–14px

Gradient accent color

Subtle glow effect matching accent

Must visually communicate forward motion.

3️⃣ PROGRESSION GRID

Use multiple glass cards containing:

Streak count (🔥 style but modern, not childish)

Level number

Badge collection preview

Completed milestones

Elements allowed:

Bars

Circular meters

Small badge icons

Mini progress rings

Everything must feel structured and disciplined, not playful.

4️⃣ BADGES SYSTEM

Badges must:

Be circular or rounded

Use subtle gradient fills

Slight glass reflection

Soft shadow

Premium metallic or neon accent depending on achievement level

No cartoon style.
No gaming childish style.

🎨 COLOR LOGIC

Accent color must represent:

Red → intensity

Blue → mastery

Green → growth

Orange → momentum

Purple → high level

Yellow → achievement

Pink → creativity

But only 1–2 accent colors visible at the same time.

The rest remains dark and calm.

✍️ TYPOGRAPHY RULES

Single modern sans-serif font

Title size: 25

Button / post text: 15

Clear spacing between sections (minimum 24px vertical rhythm)

No cramped layout

Text must breathe

🚫 FORBIDDEN

Flat design

White empty minimal startup style

Hard sharp corners

Overloaded colors

Cheap neon glow everywhere

Rigid grid with no depth

🧠 FINAL VALIDATION CHECK

Before finalizing the screen, verify:

Does this screen motivate action?

Does it clearly show progression?

Does it feel premium?

If one answer is no → redesign.
