PROMPT – PAGE PROFIL UTILISATEUR (FF)

Je veux concevoir la page profil utilisateur dans l’application FF.
Cette page doit être visuelle, claire et motivante, et mettre en avant la progression personnelle.

L’objectif du design est que le profil montre :

l’identité de la personne

sa constance

sa progression

ses communautés.

L’interface doit être mobile-first, moderne et fluide.

1. Haut de la page (Header)

En haut de la page :

À gauche

Une icône cloche pour les notifications.

La cloche est placée dans un petit rond avec fond gris "liquide milk" (style glass/liquid design).

Quand il y a des notifications :

un petit point indicateur apparaît.

À droite

Un bouton trois petits points (…)

Ce bouton ouvre les options du profil.

2. Zone identité du profil

Sous le header :

Bannière

une bannière horizontale en arrière-plan

elle occupe toute la largeur du profil.

Photo de profil

Au centre de la bannière :

la photo de profil

positionnée au centre

légèrement superposée à la bannière.

Nom utilisateur

Sous la photo :

Afficher :

@nom_utilisateur

Description

Sous le nom :

Une petite description courte (bio utilisateur).

Exemple :

Entrepreneur • Construire chaque jour • Apprendre

Nombre de followers

Sous la description :

Afficher :

nombre de followers

Exemple :

235 followers

3. Objectif principal

Sous les informations du profil :

Créer une bulle horizontale (style boudin / capsule).

Cette bulle contient :

l’objectif principal de l’utilisateur

Exemple :

Construire mon premier produit SaaS.

Design :

bulle longue

coins arrondis

style minimaliste.

4. Bloc progression personnelle

Sous l’objectif principal :

Créer une zone de statistiques personnelles.

Streak

Afficher :

La streak utilisateur.

Instruction pour le développeur :

Utiliser la même streak que celle utilisée dans les posts déjà existants dans l’application pour garder une cohérence visuelle.

Score de constance

À côté de la streak :

Créer une bulle contenant le score de constance.

Exemple :

90 / 100

Ce score représente la régularité de l’utilisateur.

La bulle doit être :

arrondie

visuellement claire

facile à lire.

Jours sur FF

À droite de cette zone :

Créer une bulle verticale rectangulaire.

Elle affiche :

le nombre de jours depuis l’inscription sur FF

Exemple :

48 jours sur FF

Cette bulle est plus verticale que les autres.

5. Barre de progression personnelle

Sous ces éléments :

Ajouter une barre de progression horizontale.

Utiliser exactement le même design que les barres de progression utilisées dans :

les cartes profil

les objectifs des communautés.

Design :

forme capsule / boudin

fond gris sombre

progression violette.

6. Diagramme d’évolution

Sous la barre de progression :

Ajouter un diagramme d’évolution personnelle.

Ce graphique montre par exemple :

la progression de l’activité

la constance

l’évolution personnelle.

Design :

graphique dans un cadre gris

coins arrondis

courbe violette

style minimaliste.

7. Communautés et connexions

Sous le graphique :

Ajouter deux sections.

Tribus

Section :

Tribus rejointes

Cette section montre les communautés que l’utilisateur a rejointes.

Afficher :

les icônes des communautés

leur nom.

Profils suivis

Section :

Profils suivis

Afficher les utilisateurs que cette personne suit.

8. Objectif UX

La page profil doit donner immédiatement une impression de :

progression personnelle

discipline

engagement.

Un visiteur doit pouvoir comprendre en quelques secondes :

qui est la personne

son objectif

sa constance

sa progression

ses communautés.

Le design doit rester :

clair

motivant

moderne

mobile-first.
