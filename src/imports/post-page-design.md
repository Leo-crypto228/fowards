MASTER PROMPT — Page Post (FF)

Tu es un product designer et développeur expert en UX/UI d’applications sociales modernes.

Ta mission est de concevoir la page complète d’un post dans l’application FF, avec une expérience fluide, intuitive et premium.

L’application cible des personnes ambitieuses, entrepreneurs et créateurs, donc l’interface doit être minimaliste, élégante et dynamique.

L’expérience doit être fluide, rapide et immersive.

1 — Ouverture du post

Dans le feed principal :

Lorsque l’utilisateur clique n’importe où sur le post, sauf :

la photo de profil

le nom de l’utilisateur

le bouton suivre

Alors :

➡️ Une nouvelle page s’ouvre avec le post complet

Cette transition doit être fluide et rapide.

2 — Structure de la page

La page contient les éléments suivants dans cet ordre.

3 — Barre supérieure

Tout en haut :

Une petite bulle avec une flèche retour.

Cette flèche permet de revenir au feed principal des posts.

Style :

minimaliste

glass effect léger

arrondi

4 — Affichage du post

Juste en dessous :

Afficher le post complet sur lequel l'utilisateur a cliqué.

Le design doit être exactement le même que dans le feed, mais en version complète.

Le post contient :

photo de profil

nom

contenu du post

éventuelles images

design premium

5 — Barre des interactions

Sous le post :

Afficher 5 interactions principales :

1️⃣ Pertinent
2️⃣ Commentaires
3️⃣ Partage
4️⃣ Visibilité du post
5️⃣ Enregistrer le post

Chaque interaction possède :

une icône

un compteur

Exemple :

Pertinent : 124
Commentaires : 32
Partages : 8
Visibilité : 1.2K
Enregistré : 12

6 — Interaction "Pertinent"

Quand l’utilisateur clique sur Pertinent :

le bouton s’active

une petite animation fluide apparaît autour de l’icône

Si l’utilisateur clique sur le compteur des pertinents :

➡️ une liste des utilisateurs qui ont mis pertinent apparaît

La liste contient :

photo de profil

nom

7 — Section commentaires

Sous la barre d’interactions :

Afficher la liste des commentaires.

Chaque commentaire affiche :

photo de profil

nom de l'utilisateur

un point

le temps depuis la publication (ex : 2h)

Exemple :

Nom • 2h

Puis le texte du commentaire.

8 — Interactions sur les commentaires

Sous chaque commentaire :

Ajouter deux actions :

bouton Pertinent

bouton Partager

Ces actions permettent d’interagir directement avec un commentaire.

9 — Scroll des commentaires

Les commentaires doivent être :

scrollables

chargés dynamiquement

rapides

10 — Barre pour écrire un commentaire

Tout en bas de la page :

Afficher une barre de commentaire arrondie (forme boudin).

Style :

coins arrondis

minimaliste

premium

Placeholder :

"Ecrire un commentaire..."

11 — Fonctionnalités d’écriture

Quand l’utilisateur clique sur la barre de commentaire :

➡️ des options apparaissent en dessous

Fonctionnalités disponibles :

GIF

Emoji

Ajouter une image

Texte en gras

Hashtags

Ces options apparaissent avec une animation douce.

12 — Fonction partage

Quand l’utilisateur clique sur Partager :

Une bulle premium style glass apparaît.

Dans cette bulle :

Titre :

Partagé à :

Ensuite afficher la liste des communautés auxquelles l’utilisateur est abonné.

Chaque communauté affiche :

photo de profil

nom de la communauté

Quand l’utilisateur clique sur une communauté :

➡️ le post est partagé dans la discussion de cette communauté

Avant de partager :

L’utilisateur peut écrire un message au-dessus du post partagé.

13 — Expérience utilisateur

L’expérience doit être :

fluide

rapide

moderne

immersive

Inspirations UX possibles :

interactions fluides

animations légères

design premium minimaliste

14 — Objectif produit

Le but est de :

encourager les interactions

augmenter le temps passé sur le post

favoriser les discussions

rendre le partage facile

créer une expérience sociale premium
