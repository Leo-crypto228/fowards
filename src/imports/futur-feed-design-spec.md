Tu es un Senior Product Designer mobile expert en produits comportementaux à fort impact identitaire.

Tu dois concevoir une application mobile iOS-first complète appelée FuturFeed (FF).

Ce n’est pas un réseau social classique.

C’est un outil social de discipline moderne.

1️⃣ POSITIONNEMENT PSYCHOLOGIQUE

FF n’est PAS une plateforme de contenu.

C’est une plateforme de :

Constance publique

Construction identitaire

Progression mesurable

Engagement envers soi-même

L’application doit transmettre :

Ambition silencieuse.
Clarté.
Force intérieure.
Modernité maîtrisée.

Pas corporate.
Pas flashy.
Pas gaming.
Pas influenceur.

2️⃣ DIRECTION ARTISTIQUE ULTRA PRÉCISE
🎨 Palette

Base sombre premium :

Bleu nuit profond (#0F172A)

Violet électrique subtil (#5B3DF5)

Gris ardoise (#1E293B)

Vert énergisant doux (#22C55E)

Accent streak :

Dégradé violet → bleu électrique

Légère lueur douce (glow maîtrisé, pas néon)

Pas de rouge agressif.

✍️ Typographie

Titre : Sans-serif moderne type SF Pro / Inter / Neue Haas Grotesk
Sous-titres : medium weight
Texte : regular
Chiffres streak : semi-bold large

Hiérarchie très claire :

H1 : 28–32px

H2 : 20–22px

Body : 15–16px

Caption : 12–13px

Beaucoup d’espace respirant.

📐 Spacing

Système 8pt grid strict.

Cartes avec padding 16–20px.
Coins arrondis 16–20px.
Ombres très légères (diffuses, pas dures).

3️⃣ NAVIGATION GÉNÉRALE

Bottom tab bar minimaliste :

Feed

Tribus

Ajouter

Profil

Icônes fines, élégantes.
Animation légère au changement d’onglet (glissement fluide).

Chaque icône active a un léger glow subtil.

4️⃣ FEED – STRUCTURE DÉTAILLÉE

Le feed doit être :

Vertical.
Fluide.
Addictif mais calme.

Inspiration structurelle :

Minimalisme de LinkedIn

Immersion verticale de TikTok

Rigueur communautaire de Reddit

🧱 Carte d’avancement (ultra détaillée)

Chaque carte contient :

Header

Avatar rond (40px)

Nom

Objectif principal en sous-titre

🔥 Badge streak à droite (visuellement fort)

La streak doit :

Être légèrement animée au scroll (micro glow)

Avoir un effet subtle pulse si > 30 jours

Bloc central

Section “Avancement du jour”

Label discret

Texte principal

Icône type d’avancement (micro, majeur, réflexion)

Sous ce bloc :

Mini barre de progression animée.
Animation douce de remplissage.

Section preuve

Court texte libre.
Maximum 4 lignes visibles.
Bouton “Voir plus” animé.

Interactions

Bouton “Pertinent”
Bouton “Conseil”

Hover → élévation légère + glow subtil
Pressed → micro réduction 95% + vibration légère
Active → état visuel distinct (bordure lumineuse fine)

Aucun compteur visible.
Mais animation interne de feedback.

5️⃣ CRÉATION D’AVANCEMENT

Écran très épuré.

Fond légèrement plus lumineux que le feed.

Composants :

Sélecteur d’objectif (dropdown animé)

Sélecteur type d’avancement (3 cartes cliquables)

Champ texte minimal

Bouton principal large

Bouton “Valider ma constance”

Quand appuyé :

Animation fluide.
Transition douce.
Message central :

“Constance maintenue. Continue.”

Effet de légère vibration + glow streak mis à jour.

6️⃣ PROFIL – CENTRE NÉVRALGIQUE

Le profil est un dashboard public.

Inspiration :

Dashboard moderne type produit SaaS

Élégance premium type produit tech haut de gamme

Structure :

Section haute

Streak en très grand (élément central)
Objectif principal
Badge constance

Animation au chargement :

La streak se remplit dynamiquement.

Section progression

Graphique simple (ligne douce animée)
% progression
Historique des jours validés en grille

Chaque jour validé = point lumineux subtil.

Hover → affiche détail.

Section analytique

Nombre total jours constants

Progression par objectif

Graphique activité

Prochaine étape définie

Tout doit être fonctionnel.

Chaque bouton doit :

Avoir un état hover

Un état pressed

Un état active

Un état disabled

Micro-animation fluide à chaque interaction.
